2-3.선 그리기

#include <customwindow.h>

int main() {
    Window* window = new Window();
    DRAW_TOOL* draw = new DRAW_TOOL(window);
    while (window->window_running()) {
        draw->start();
        draw->Line(RGB(0, 0, 0), 10, 10, 100, 0);
        draw->end();
    }
    delete draw;
    delete window;
}
!delete를 쓰지 않아도 오류가 나지는 않지만 공간을 게속 차지할 수 있기 때문에 지워주는게 더 좋다.

3-1.버튼 만들기

CREATE_TOOL* button = new CREATE_TOOL(window, CLASSNAME::BUTTON, "button text", 10, 10, 100, 20, id);
버튼 기능은 아직 재대로 구현 되지 않음.
버튼 기능을 만드려면 네모를 그린 뒤 커서를 감지해야 함.
CREATE_TOOL도 다른것들과 마찬가지로 마지막에는 delete로 지워줘야함

3-2.텍스트 박스 만들기

CREATE_TOOL* textbox = new CREATE_TOOL(window, CLASSNAME::TEXTBOX, "", 10, 10, 100, 20);

3-3.텍스트 박스 읽기

textbox.gettext(buff);

4-1.마우스 입력

input::mouse();
괄호 안에 window변수를 넣어 주면 윈도우기준 커서 위치를 알려준다.

4-2.마우스 입력2

if (GetAsyncKeyState(0x0001) & 0x8000) {
    POINT cursor = input::mouse(window);
    //code
}
GetAsyncKeyState대신 input::key를 사용해도 된다.
0x0001은 좌클릭,0x0002는 우클릭이다.
window부분에 아무것도 입력하지 않으면 전체 창을 기준으로 좌표를 알려준다.

4-3.텍스트 입력

if (input::key(VK_RETURN)) {
    //code
}
input::key의 두번째 인수에 input::mouse처럼 window변수를 넣어 주면 현제 그 윈도우가 최상단에 있을때만 감지를 한다.
